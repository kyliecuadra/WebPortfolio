# Portfolio Website

A modern, recruiter-focused developer portfolio built to showcase professional experience, technical expertise, GitHub projects, certifications, and career achievements. The application is designed with a strong emphasis on performance, accessibility, responsiveness, and search engine optimization, providing an engaging experience across desktop, tablet, and mobile devices.

---

## Architecture & Technologies

### Frontend
- HTML5 with semantic and accessible markup  
- CSS3 using custom design tokens, Flexbox, Grid, and modern animations  
- Vanilla JavaScript (ES2020+) with a modular architecture  
- Responsive design optimized for all screen sizes  
- Dark and Light theme support with persistent user preferences  

---

## User Experience Features
- Animated hero section with dynamic role rotation  
- Scroll-based reveal animations powered by Intersection Observer  
- Interactive project filtering system  
- Animated skill proficiency indicators  
- Mobile-first navigation with responsive menu controls  
- Smooth scrolling and active section tracking  

---

## GitHub Integration
- Real-time GitHub API integration  
- Automatic retrieval of repositories and profile statistics  
- Repository language analysis and technology breakdown  
- Dynamic project showcase generated from GitHub data  
- README-based project descriptions for enhanced presentation  

---

## Performance & Accessibility
- Framework-free implementation for minimal bundle size  
- Lazy-loaded animations and optimized rendering  
- Semantic HTML structure and ARIA accessibility support  
- Keyboard navigation support  
- Reduced-motion compatibility  
- SEO optimization using Open Graph, Twitter Cards, and Structured Data (JSON-LD)  

---

## Design System
- Centralized design tokens for colors, typography, spacing, and component styling  
- Reusable UI components following a consistent visual language  
- Modern glassmorphism-inspired interface with professional developer branding  

---

## Purpose

This portfolio serves as a centralized platform for presenting software engineering experience, enterprise application development expertise, backend engineering projects, AI and computer vision solutions, and open-source contributions. It is designed to help recruiters, hiring managers, and potential clients quickly evaluate technical capabilities and professional accomplishments.
